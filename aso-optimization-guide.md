## 🎯 FUNDAMENTOS DE ASO

### O Que É ASO?
App Store Optimization é o **SEO para apps**. 70% dos usuários descobrem apps via busca nas lojas.

### Fatores de Ranking

| Fator | Peso | Controlável? |

## ⭐ ESTRATÉGIA DE REVIEWS

### Objetivo
- **Meta**: 4.5+ estrelas (ambas as lojas)
- **Quantidade**: 100+ reviews (primeiro mês)

### Timing do Popup

```typescript
// Quando pedir review?
const shouldAskForReview = (user: User): boolean => {
  return (
    user.daysActive >= 7 && // Pelo menos 7 dias de uso
    user.adherenceRate >= 80 && // Alta adesão (usuário satisfeito)
    user.medicationsTaken >= 20 && // Usou bastante
    !user.hasReviewed && // Ainda não avaliou
    !user.reviewDismissed // Não recusou antes
  );
};
```
